This project requires the "Configuration Section Designer" extension for Visual Studio. (http://csd.codeplex.com/)

To regenerate PalMonConfigSection.cs and PalMonConfig.xsd, build this project and:
	* Copy the contents of ConfigurationSection.csd.cs into \PalMon\PalMonConfig\PalMonConfigSection.cs
	* Copy the contents of ConfigurationSection.csd.xsd into \PalMon\Resources\PalMonConfig.xsd
